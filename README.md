# Bookmarker for Nextcloud

## Caution

This Chrome/Firefox/... extension is still in early development stage but is fully operational. Nevertheless, it might crash your computer, delete your doctoral thesis, or even kill your cat! 😽

# What it's for?

Fast and easy bookmarking with your Nextcloud installation.

##### But there's already a bookmarklet available!

Although it does its job it's a debris. A small popup in the browser with some more (future) fetures fits more into the daily work flow.

# Requirements

- [Nextcloud installation](https://nextcloud.com/)
- [Bookmark Addon](https://apps.nextcloud.com/apps/bookmarks)

# Installation

This extension is still in early development and hasn't been released to any extension/addon site yet. You have to install the unpacked code. It's a bit complicated but it sonn will change.

## Chrome

- Download the code.
- Go to the three dots on the right and open the menu.
  -> More tools
  -> Extensions
- In the upper right corner enable "Developer mode".
- On the left press "Load unpacked" and navigate to the download directory.
- Select directory

The extension will not appear automatically. You have to click the "Extension puzzle piece" int the toolbar and pin "Bookmarker for Nextcloud".

## Firefox

You can simply download this addon from box.com: [Bookmarker for Nextcloud](https://app.box.com/s/6pkcs8g3fb6dq96a5py2dhllj6v5dx6u)

# Granting access

If you click on the icon for the first time, a popup window will open and ask you for the Internet address (URL) of your Nextcloud installation.

![hostname](https://elektroelch.de/githubpics/Nextcloud-00-hostname.png)

After entering the correct address your default web browser will open a tab and display the "Nextcloud login flow v2" as shown in the image below.

![webflow](https://elektroelch.de/githubpics/collection.png)

After granting access your ready to save your first bookmark.

# Usage

Saving a bookmark is straightforward. Click on the Icon in the toolbar of your browser and a popup will open:

![popup](https://elektroelch.de/githubpics/popup.png)

Fill in the information and click "Add" and the bookmark will be stored stored. As easy as that.
If the bookmark is already stored, the popup will inform you and change the popup accordingly.
